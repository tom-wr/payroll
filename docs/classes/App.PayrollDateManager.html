calculateSalaryDateForMonth()

calculateSalaryDateForMonth(integer  $month) : string

Calculates the salary date for a given month.

The salary date should be the last day of the month. If the date falls on a weekend the date is offset to the previous Friday.

Parameters

integer $month

The month's index i.e. 1 refers to January

Returns

string —

The date calculated for the salary day.
